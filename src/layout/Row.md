Example:

```js
<Row>
  <p>Item</p>
  <p>Item</p>
  <p>Item</p>
</Row>
```
Items aligned to the right
```js
<Row hAlign="right">
  <p>Item</p>
  <p>Item</p>
  <p>Item</p>
</Row>
```

Items that wrap
```js
<Row wrap>
  <p style={{minWidth: "50%"}}>Item</p>
  <p style={{minWidth: "50%"}}>Item</p>
  <p style={{minWidth: "50%"}}>Item</p>
</Row>
```
