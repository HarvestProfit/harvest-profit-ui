Example:

```js
<Column>
  <p>Item</p>
  <p>Item</p>
  <p>Item</p>
</Column>
```
Items aligned to the right
```js
<Column hAlign="right">
  <p>Item</p>
  <p>Item</p>
  <p>Item</p>
</Column>
```

Items vertically gapped (space-between)
```js
<Column vAlign="gap" style={{height: 400}}>
  <p>Item</p>
  <p>Item</p>
  <p>Item</p>
</Column>
```

Items vertically padded (space-around)
```js
<Column vAlign="pad" style={{height: 400}}>
  <p>Item</p>
  <p>Item</p>
  <p>Item</p>
</Column>
```
